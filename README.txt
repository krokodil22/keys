# RU Keyboard Trainer (уровни 1–5 + бонус ∞) — v2

Запуск:
1. Скачайте и распакуйте архив.
2. Откройте `index.html` в браузере.

Обновления в v2:
- Буква при смене уровня/рестарте всегда появляется у правого края — стартует «с начала пути».
- До нажатия ПРОБЕЛА буква не двигается и не «ползает» — позиция фиксирована.
- Всё остальное: уровни 1–5, бонус ∞, сохранение прогресса, SVG‑сердечки, подсветка клавиш.
